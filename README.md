# Student Score Predictor

A machine learning project that predicts student exam scores based on:

- Hours Studied
- Attendance
- Tutoring
- Gender
- Region

## Model

- Algorithm: Linear Regression
- R² Score: 0.9087

## Features

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Model Training
- User Input Predictions

## Technologies

- Python
- Pandas
- Matplotlib
- Scikit-Learn
